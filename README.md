# Startup OS – Co-Founder Dashboard

Co-Founder Dashboard für Raphael & Martim (HSG St. Gallen).

## Features

- Ideen-Board mit KI-Analyse (Claude Sonnet)
- Ideen-Abstimmung (geheim, beide müssen abstimmen)
- Entscheidungs-Matrix & Hypothesen-Log
- Market Sizing mit KI-Marktschätzung (TAM/SAM/SOM)
- Roadmap & Milestones
- Weekly Planner (Tasks pro Person)
- KPI Dashboard
- Fundraising Pipeline mit KI-Investor-Analyse
- Pitch Tracker mit KI-Feedback-Analyse
- Competitor Intelligence & Network
- AI Assistent (Claude Sonnet, voller Projekt-Kontext)
- Pitch Deck Export (automatisch aus Plattformdaten)
- One-Pager Export
- Firebase Real-Time Sync (beide sehen Änderungen sofort)

## Deployment

Automatisch via Netlify bei jedem Push auf `main`.

## Setup

1. Anthropic API Key im AI Assistent eingeben (wird nur lokal gespeichert)
2. Firebase ist bereits konfiguriert (startup-os-hsg)
